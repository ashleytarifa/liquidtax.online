how did i started drawing?

i always wanted to be an artist, but i could never draw anything. after developing a passion for indie games and pixel art, i decided to give it a try and drew something very simple. however, i saw potential and ended up creating my first landscape. i liked it and have continued drawing ever since. even with the challenges of daily life (school and work), i still make some time to do what i love S2

do i do something else besides drawing?

i know some web developing (HTML and CSS) so i code some little silly stuff here and there, but i intend to be a programmer or game dev one day!

i also collect old consoles, play some games, listen to a LOT of music, and im now building my own pc! im almost finishing it tho :3

purrgatory (portuguese brazilian translation)

Purrgatory is a visual novel about passing time in an afterlife where nothing really matters. Explore a whimsical point-and-click world, get to know its downtrodden inhabitants, and remember — furever's not so bad!

purrgatory was the first game i played when i created my steam account, so this game has an emotional value for me
one day (in like 2023) the dev (niv) was looking for volunteer translators to translate his game, i've never translated a game before, but i saw an opportunity at that time and it worked! the game is on fully ptbr now, and i intend to translate other games later on, who knows?
